## API 错误日志

  <br>

控制台-》多媒体彩信-》点击左侧栏“[API 错误日志](https://www.mysubmail.com/console/mms/errors)”，即可进入彩信 API 错误日志页面。可以帮您查看 API 彩信的错误情况。

 <br>

### **错误日志详情**

<br>

可以设置日期范围查看 API 错误日志。
错误日志详情包括 AppID、API、错误码、日期、IP 来源以及错误描述等。
